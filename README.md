A personal challenge repository where I solve at least **one LeetCode problem every day for 1 month** and upload the solution consistently.

## 🎯 Goal
The main purpose of this challenge is to:

- Build consistency in problem-solving  
- Strengthen Data Structures & Algorithms concepts  
- Improve coding speed and logical thinking  
- Prepare for coding interviews and competitive programming  

## 📂 Repository Structure
Each file contains:

- Problem Name / Number  
- Approach / Logic Used  
- Optimized Solution in Java
